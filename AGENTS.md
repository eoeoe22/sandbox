# 이 프로젝트는 웹 브라우저 기반 샌드박스 파티클 시뮬레이터임.
1순위 : 재미
2순위 : 편의성
3순위 : 과학적 고증

# 워크플로우

1. 새 브랜치에서 작업 수행
2. 작업 후 1차 Push 및 gh CLI로 Pull Request 생성
3. 서브에이전트 호출을 통해 코드리뷰
4. 리뷰가 no issues로 통과 시 call_user 도구 사용 / 리뷰에서 지적사항이 있다면 조치후 리뷰 재시도 (no issues 확정 까지 반복)
5. 사용자의 승인 대기
6. Cloudwiki에서 "Web sandbox 프로젝트" 문서 업데이트. 해당 문서는 즉시 반영으로 편집을 권장함.
