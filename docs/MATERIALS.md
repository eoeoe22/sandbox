# 물질 도감 (Materials)

전체 물질과 핵심 상호작용 요약. 팔레트는 아래 카테고리 탭으로 묶인다(각 물질의
`category` 필드, 생략 시 `phase` 에서 자동 유도).

> `Eraser`(지우개), `Ember`(불씨), `Spark`(스파크), `Nuclear Ray`(핵 광선),
> `Heat Ray`(열선)는 팔레트에 없다: 폭발 파편/전기 펄스/임계 핵 광선/Laser가 쏘는
> 레이저 빔으로 시뮬레이션이 스스로 생성하는 일시적 입자다. (열선을 쏘는 **Laser**
> 방출기 자체는 전기 탭에 있다.)

## 🪨 고체 (Solid)

| 물질 | 거동 / 상호작용 |
|---|---|
| Wall | 파괴 불가 경계. 폭발·산·Void 모두 못 뚫음. **온도 시스템 완전 배제**(자체 온도 없음, 열전도 0 — 완벽 단열 경계). |
| Stone | 정적 장벽. 1100°↑ → Lava 로 용융. |
| Glass | 투명. 내산성. 1250°↑ → Molten Glass 로 재용융. **충격파에 깨짐** — 완전히 부수지 못하는 약한 충격(Gunpowder 폭발·약한 폭발 충격파·Woofer 충격파)에 노출되면 그냥 버티지 않고 **Broken Glass(가루)로 부서진다**(그래도 파동은 여전히 막아 뒤를 가림 — `shatterId`, blast.ts). 더 강한 폭발은 종전대로 통째로 소멸(크레이터 섬광). **Heat Ray(레이저) 투과 시 핑크빛 광선이 유리 내부에 시각적으로 투과함**. |
| Concrete | Cement + 물이 굳은 결과물. 강하지만 폭발·산에 취약. Cement 가루보다 어두운 색상(`rgb(110, 112, 118)`)에 가루처럼 낮은 dynamic range의 개별 파티클 틴트가 적용됨. |
| Diamond | 최고 열전도. **모든 폭발·고온·저온·산에 면역**. 방폭·전열 구조물. 2x2 체커보드 패턴(Mesh보다 두꺼움)으로 렌더링. 유일한 예외: 우라늄 임계 핵 광선(Nuclear Ray)은 다이아몬드도 뚫는다. |
| Amber | 송진(Resin)이 시간이 지나 굳어 만들어지는 반투명 황금빛 고체. 장식·봉인용("호박 속 곤충" 밈). 여전히 나무 수지라 **가연성**(느리게 탐). **불타는 동안**(700°↑) 낮은 확률로 일부만 다시 Resin 방울로 **용해** — 전체가 녹지 않고 불타는 덩어리 중 대략 1/10 정도만 끈적하게 녹아 흘러내린다(나머지는 나무처럼 탄 뒤 재로). 가역 순환: 식으면 굳고, 타면서 일부가 녹는다. |
| Aerogel | **완전 단열재**(열전도 0) — 열이 전혀 안 통해 용암/얼음 실험 격벽으로 최적. 단, **방폭 아님**: 열만 막고 힘은 못 막아 폭발엔 일반 고체처럼 부서지고 산·테르밋에도 당한다. |
| Rust | **녹(고체)**. 철(Iron)이 소금물(Saltwater)과 접촉할 때 표면(깊이 2칸 이내)에서 천천히 생성되는 산화철 고체. **전기 절연체**(Clean Iron과 달리 전기를 안 통함). 1538°C 이상 가열 시 50% 확률로 Slag, 50% 확률로 Molten Iron Ore로 융해. |
| Gallium | **손 위에서 녹는 금속**. 녹는점이 상온(20°) 바로 위인 **30°**밖에 안 돼, 조금만 데워도(불씨·따뜻한 이웃·근접 복사열) 스르르 **Liquid Gallium**으로 녹아 반짝이는 웅덩이가 된다. **전기 도체**(스파크 전달). 금속이라 열전도가 좋아 살짝 닿은 열이 막대 전체로 퍼져 통째로 녹는다. |

## 🏖️ 가루 (Powder)

> **밀도 부력(모든 가루 공통):** 모든 가루는 밀도가 있어, 자신보다 무거운 액체에
> 파묻히면 그 액체를 밀어내며 스스로 수면 위로 떠오르고, 반대로 자신보다 가벼운
> 액체에는 그냥 가라앉는다 — 물에 파묻힌 Ash 가 부글부글 떠오르는 것과 똑같은
> 메커니즘을 팔레트의 **모든** 가루가 공유한다(더 이상 Ash·Sawdust·Limestone·Coal
> Powder 같은 소수 예외 전용이 아니다). 유일한 예외는 **제련용 Limestone**으로,
> Molten Iron Ore·Slag에서만 밀도 대신 재질별 특수 규칙을 따르며 눌러앉는다(계속
> 반응 중인 층이라 뜨면 안 되므로) — 다 끝난 Molten Metal에서는 그것도 그냥
> 밀도로 떠오른다. **제련용 Coal Powder**도 이 예외 코드 경로를 여전히 거치지만
> 실제로는 아무 효과가 없다: 제련 밀도 재서열(아래 **제련** 탭 참고) 이후 자기
> 밀도가 이미 Molten Iron Ore·Slag보다 무거워져서, 예외가 없어도 그냥 가라앉기
> 때문이다.

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Sand | 5 | 낙하·퇴적. 1250°↑ → Molten Glass. |
| Salt | 5 | 물에 용해 → Saltwater. 초고온(800°↑) → 용융염(Molten Salt). |
| Soda | 5 | 베이킹소다 — **분말 소화기 + 산 중화제**. 인접한 불을 즉시 소화하고 **타고 있는 연료도 식혀서 진화**(소화 작업마다 조금씩 소모 — 김을 내며 사라짐). Acid 접촉 → **Saltwater 로 중화**(소금+물, 실제 반응 산물) + 거품. Acid Vapor 도 흡착 제거. 150°↑ 가열 시 분해되어 김으로 소멸. |
| Dirt | 5 | 물 흡수 → Mud. 생태계의 지반 층. |
| Cement | 5 | 물 접촉 → Concrete 로 경화(물 소모). |
| Metal Powder | 7 | 드럼통이 폭발로 부서질 때 나오는 산탄형 철가루. 낙하·퇴적하며, 철 녹는점(1200°) 이상으로 가열하면 Iron 과 마찬가지로 Molten Metal 로 녹는다. Sand·Salt 급 광물 가루(5)보다 무거워 가라앉지만, 고체 Iron 블록만큼 무겁진 않다. 소금물 접촉 시 안쪽까지 서서히 **Rust Powder**로 부식. |
| Rust Powder | 5.5 | **녹가루(가루)**. 금속 가루(Metal Powder)가 소금물에 스며들어 부식되거나 철 부식 시 드물게(20%) 발생하는 가루. 1538°C 이상 가열 시 50% 확률로 Slag, 50% 확률로 Molten Iron Ore로 융해. |
| Broken Glass | 1000 | **깨진 유리(가루)** — 유리판이 충격파에 깨져 나오는 형태(Glass 항목 참고). 나머지는 Glass 와 동일: 내산성, 낙하·퇴적하며 1150°(재용융 유리라 Sand보다 낮은 융점)↑ 가열 시 Molten Glass 로 녹아 다시 식으면 맑은 Glass 판이 된다. 밀도가 Glass(1000) 그대로라 모든 액체·가벼운 가루 밑으로 가라앉는 무거운 유리 알갱이. |
| Soap | 4 | 물·소금물에 닿으면 녹아 **Soapy Water**로(닿은 물 쪽도 함께 비눗물이 됨). 유기물이 아니라 바이러스 침식에 면역 — 깨끗한 방역선. |
| Ash | 1.5 | 연소 잔재. 아주 가벼워 물에 뜬다. 가루가 흩날리듯 좌우로 흔들리며 천천히 떨어진다. 물에 파묻혀도 고체처럼 눌려 있지 않고 물을 밀어내며 스스로 수면 위로 떠오른다(물 붓기로도 위에 안 쌓이고 부글부글 뚫고 올라옴). |
| Sodium | 2.5 | **물에 닿으면 격렬 반응**: 불꽃+수소 방출+발열. 소량은 물 위를 스치며 타는 화구, **뭉쳐 쌓으면 물 접촉 시 덩어리째 폭발**(양에 비례). 물보다 가벼워 수면에서 반응. |
| Sugar | 3.65 | 설탕. 불에 잘 타는 가연성 가루. **가열(200°↑)하면 캐러멜화→탄화(→재)**. 물엔 안 녹음(효모용) — **효모+물과 만나면 발효**(→Alcohol+CO₂). 소금(5)보다 가벼운 유기물이라 물엔 가라앉지만 Saltwater 이상 액체에는 뜬다. |

## 💧 액체 (Liquid)

> **어는점(경량 방식):** 모든 일반 액체는 각자의 어는점 이하로 냉각되면 **제자리에서 굳어**
> — 흐르지 않고 고체처럼 밀도 치환도 막으며 서리색으로 표시 — 데우면 다시 흐른다. 별도
> 물질로 바뀌지 않는다(물만 예외 — Snow/Ice 로 상전이). 물·소금물은 **전기 도체**이기도 하다.

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Water | 3 | 100° 끓음→Steam, 0°↓ 얼음/눈. **전기 도체**(약함 — 스파크가 몇 칸 못 가 감쇠 소멸). 전기 통하면 소량의 수소·산소 생성(전기분해). |
| Saltwater | 4 | 물보다 무겁게 가라앉음. 끓으면 소금 석출. -18°↓ 동결. **전기 도체**(약전해질 — 스파크가 물보다 멀리 전달). 전기분해로 소량 수소·산소. |
| Acid | 3 | 비내산성 고체·가루 부식. 끓으면 Acid Vapor. -20°↓ 동결. |
| Mercury | 9 | 가장 무거운 일반 액체(Molten Uranium(10) 다음). **전기 도체**(스파크 전달). -38°↓ 동결. **357°↑ 끓음 → Mercury Vapor**(식으면 다시 응결). 단색 렌더(배경 틴트 샘플링 제외). |
| Liquid Gallium | 6 | Gallium 이 녹은 액체 금속(Mercury보다 가벼워 그 위에 뜬다). **전기 도체**(스파크 전달). 표면장력으로 방울지고 단색 렌더. **28°↓로 식으면 다시 고체 Gallium 으로 굳는다**(녹는점 30°와 약간의 이력차). |
| Honey | 3.5 | 점성 높아 천천히 흐름. 느리게 타는 연료. **물과 섞임**(Acid처럼 상호확산). 5°↓ 굳음(결정화). |
| Alcohol | 1.9 | 가장 가벼운 연료. 순식간에 옮겨붙어 확 탄다. -80°↓ 동결. **소독제**: 닿은 바이러스를 직접 없앰(H2O2와 달리 확산 없이 **닿은 부분만**). |
| Mud | 5.5 | 젖은 흙. 마르면 Dirt 로 굳음. -3°↓ 동결. |
| H2O2 | 3.4 | 과산화수소. 스스로 분해하며 **산소 방출**(→불 가속제)+물로 환원(환원 속도 느림 — 웅덩이가 오래 감). 가열·촉매 시 분해 가속(불 근처면 산소 폭주). **바이러스 부식 살균**: 닿은 바이러스를 부식시키며 **랜덤한 방향으로 우둘투둘** 파고든다(깔끔한 사각 경계 없음). 살균 1회마다 H2O2가 소모(→물)돼 **뿌린 양에 비례**한 만큼만 번지므로, 극소량으론 대량 군집을 다 못 지운다(무한 확산 없음). |
| Resin | 2.6 | 송진(물에 뜸). 끈적한 가연성 액체지만 잘 붙진 않음 — 한 번 붙으면 **천천히 번지며 오래 탄다**. 시간이 지나면 **굳어 Amber(호박)로 경화**(400° 미만에서만). 데우면 굳기 전에 탄다. **가역**: 불타는 Amber 도 낮은 확률로 일부가 다시 Resin 으로 녹아(700°↑) 순환한다. |

## 💨 기체 (Gas)

| 물질 | 거동 / 상호작용 |
|---|---|
| Steam | 상승 후 응결 → Water. |
| Smoke | 상승 후 확률 소멸(수명 짧음 — 예전의 1/3, ~0.6초). |
| Acid Vapor | 상승하며 부식, 응결 → Acid. |
| Mercury Vapor | Mercury 를 끓는점(357°) 이상 가열하면 발생. 상승·확산하다 식으면 응결 → Mercury(Steam↔Water 와 같은 순환). |
| Oxygen | 불 접촉 시 자신도 **Blue Flame**으로 확 번짐(가속제 — 일반 불이 아니라 산소 문 화염 자체가 스폰). 수소와 함께 연소하면 수증기(→물) 생성. |
| Hydrogen | 가장 가벼운 **가연성 기체**(폭약 아님). 점화 시 연소해 불로 번짐. **Oxygen 과 만나 점화되면 물(수증기) 생성**(2H₂+O₂→2H₂O). |
| CO2 | **무거운 기체** — 바닥에 깔려 고이고 낮은 곳을 채운다(최초의 하강 기체). 불꽃을 즉시 끄고 **타는 연료(석탄·나무 등)까지 식혀** 진화하므로 근접 인화로 되살아나지 않는다(Soda식 냉각 소화 — 퍼지는 기체라 광역으로 작동). Dry Ice 승화물(차가운 CO₂ 안개). 서서히 공기로 소멸. |
| Chlorine | 무겁고 **독성** 기체 — 바닥에 깔려 생물(Plant·Virus·Yeast·Seed·Slime)을 살상. 서서히 소멸. (원래 표백제+산으로 생성 — 표백제 미구현이라 현재는 직접 배치) |
| Helium | **초경량** — 모든 기체 위로 상승해 천장에 고임. 무반응(풍선 개그). |

## 🔥 불·열 (Fire / Heat)

| 물질 | 거동 / 상호작용 |
|---|---|
| Fire | 가연물 점화, 물에 소화. |
| Blue Flame | 초고온. 돌을 녹여 Lava 로. **연기 미생성**(깨끗한 절단 토치). |
| Lava | 용융 암석. 식으면 Stone. |
| Molten Metal | Iron 이 녹은 것. 식으면 Iron. 밀도 8 액체(제련 3상 중 가장 무거워 노 바닥에 고인다). |
| Molten Glass | Sand 가 녹은 것. 식으면 Glass. |
| Molten Salt | Salt 가 녹은 것(800°↑). 식으면 Salt. |
| Coal / Wood / Sawdust | 고체 연료(느림→빠름 순: Coal < Wood < Sawdust). 액체 연료(원유·휘발유 등)는 아래 **석유** 탭 참고. Wood·Sawdust는 소모될 때 일부 셀이 Fire로 사라지는 대신 소량 확률로 **Ash**를 남긴다(15%). Sawdust는 물보다 가벼운 **가벼운 가루**라 물에 파묻혀도 고체처럼 눌려 있지 않고 스스로 수면 위로 떠오른다(Ash와 동일 메커니즘). |
| White Phosphorus | 백린(가루, 밀도 3.2). **공기 중 자연발화** — 노출되면 백색 화염+짙은 백연(연막)을 뿜고 주변 가연물을 점화하며 계속 낙하·연소. **물속에선 안전**(잠기면 불활성, 담그면 소화) — 물보다 무거워 가라앉으므로 물에 담그면 확실히 잠긴 채 유지된다. |

> **산소 송풍(forced draught):** 모든 연료는 연소 중 인접한 **Oxygen** 1칸당 화염 온도가
> +250°(상한은 **Blue Flame과 동일한 1800°**) 오르고, 빨려든 산소는 확률적으로 소모된다.
> 대부분의 맨 연료불은 800°라 산소를 불어넣어야 1050°(1칸)·1300°(2칸)·1550°(3칸)·
> 1800°(4칸+, Blue Flame급)까지 오르지만, **Coal/Coal Powder는 예외적으로 맨불 자체가
> 1300°**로 핀 된다(산소 없이도 철(1200°)까지 녹임 — 아래 **제련** 탭 참고). 이 온도는
> 돌(1100°)도 함께 녹이므로 노·용기는 **Diamond·Heatpipe**로 짓는다(둘 다 열전도
> 최상급이라 열이 장입물로 잘 전달되고 녹지 않는다). 산소가 조금이라도 섞이면 **온도만
> 오르는 게 아니라 감싸는 불꽃·소진 시 붕괴하는 불꽃 자체가 Fire 대신 진짜 Blue Flame로
> 바뀐다** — 산소 문 화염은 Blue Flame의 암석 용융·긴 수명까지 그대로 물려받는 진짜
> 절단 토치가 된다(단, CO2·Soda의 "인접 Fire 즉시 소화"는 정확히 Fire만 겨냥하므로 이
> 불꽃엔 안 통한다 — Blue Flame이 원래 그렇듯, 대신 연료 자체를 식히는 방식으로는 여전히
> 꺼진다). 연료 종류 무관이라 "산소+아무 연료 = Blue Flame"이 성립하며, 석탄불에 산소까지
> 불면 **2칸만으로** 상한(1800°)에 닿는다(맨불 1300°가 이미 3칸치 화력이라).

## ⚒️ 제련 (Smelting)

철광석에서 철을 뽑는 **제련 공정 놀이**. 정체성은 분별증류의 "직화=연소 / 간접열=증류"
대비를 잇는 **"그냥 녹이면 쓸모없는 슬래그(실패) / 탄소(석탄)와 함께 구우면 철(성공)"**
이다. 공정은 **2단계** — ① **철광석을 녹인다**(일반 Fire로도 충분, 850°↑), ② 녹은
**용융 철광석 웅덩이에 석탄가루를 뿌린다** → 철. 녹이기만 하고 석탄을 안 넣으면 용융
광석이 식어 **쓸모없는 Slag로 굳는다** — "열만으로는 슬래그, 석탄과 함께여야 철".

**1단계 용해:** 철광석(가루)은 850° 이상이면 **용융 철광석**(빛나는 액체)으로 녹는다.
Fire 셀은 ~1000°라 **불만 충분히 대면 녹고**, 석탄불은 1300°로 더 확실히 녹인다(Lava·
Blue Flame·히트 브러시도 물론 녹인다) — 단 **광석에 직접 닿은 Coal Powder는 연소하지
않고 환원제로만 작동**하므로(아래 핵심 항목 참고, 고체 Coal은 이 차폐가 없어 직접 닿아도
그냥 타서 녹인다), 웅덩이를 만드는 것은 대개 한 칸 이상 떨어져서 타는 석탄이 전도로
데우는 열이다. **2단계 환원:** 용융 웅덩이에 닿은 **석탄가루/
석탄**이 광석을 환원한다 — **석탄 1칸이 광석 1칸을 즉시(~4틱) 환원**하고 자신은 소모되며
연기(CO₂)를 낸다. 완료 시 **Molten Metal**(쇳물)이 되고 식으면 **Iron**으로 굳는다. 쇳물이
제련 3상 중 가장 무거워 **노 바닥으로 가라앉아** 고이고, 가벼운 슬래그가 그 위로 떠 층이 갈린다
(실제 고로에서 슬래그 아래에 쇳물이 고이는 것과 같다).
**Limestone** 플럭스가 철 수율(0.70→0.95)을 올린다.

**핵심(용해·혼합 문제 해결):** ① 환원은 **발열 반응**이라 산물이 뜨거운 쇳물(≥1450°)로
나와 **가라앉는다** — 예전처럼 산물이 그 자리에서 고체 철로 굳어 웅덩이를 봉해버리는 크러스트가
생기지 않는다(쇳물이 웅덩이 아래로 흘러 빠져나간다). ② **광석에 직접 닿은 석탄은 연소하지 않고 환원제로만 작동**한다(환원 분위기) — 그래서
**석탄을 광석과 미리 섞거나 층으로 쌓아 가열해도 석탄이 먼저 타 없어지지 않고** 광석을 환원한다.
광석에서 한 칸 이상 떨어진 석탄은 정상적으로 타서 열을 낸다. ③ **용융 웅덩이에 뿌린 석탄은
웅덩이 속으로 가라앉아 섞이므로** 표면만이 아니라 전체가 환원된다 — 석탄가루가 웅덩이보다
무거워(아래 **밀도 층화** 참고) 원래도 가라앉지만, 뿌린 즉시 확실하게 한 칸 더 밀어 넣어
표면에서 스치기만 하다 마는 일 없이 깊숙이까지 고르게 저어 넣는 셈이다.

**두 가지 방법 모두 작동:** (A) 광석을 녹인 웅덩이에 석탄가루를 뿌리면 가라앉아 섞이며
환원(얕은 웅덩이 수율 ~65%), (B) 석탄가루와 광석을 미리 섞거나 층으로 쌓고 밖에서 가열하면
석탄이 타지 않고 녹은 광석을 환원(장입 방식, 수율 ~60%). 석회석을 더하면 ~87%.

**밀도 층화:** 쇳물(8) > 석탄가루(7.5) > 용융 철광석(6.5) > 슬래그(5.75) > 석회석(5) —
게임플레이 우선 순서(실제 석탄 비중은 이보다 훨씬 가볍지만, 재미를 위해 웅덩이를 뚫고
가라앉도록 조정했다). 슬래그를 진흙(Mud, 5.5)과 겹치지 않는 5.75로 잡은 것도 의도적 —
Thermite(밀도 5.6, **폭발** 탭)가 "진흙까지는 가라앉고 Slag 이상에서만 뜬다"로 이미
캘리브레이션돼 있어, 슬래그가 5.5까지 내려가면 진흙과 같아지며 Thermite가 슬래그도
뚫고 가라앉아 버렸을 것이다. 뿌린 석탄은 스스로 웅덩이를 뚫고 가라앉으며 지나는 길의
광석을 환원하고, 환원된 쇳물은 그보다도 더 무거워 바닥으로 가라앉으며, 폐기물 슬래그와
석회석 플럭스는 가벼워 위로 떠오른다 — 노의 수직 존이 저절로 생긴다(실제 고로처럼
쇳물이 맨 아래, 슬래그가 그 위에 뜬다).

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|

| Limestone | 5 | 선택적 **플럭스**(백악색 가루). 인접 환원 중 용융 광석의 철 수율을 0.70→0.95로 올리고 소모된다(하소 연출). 없어도 제련은 되지만 슬래그가 섞인다. 제련 5종 중 **가장 가벼워**, Molten Iron Ore·Slag에는 자기 밀도만으로는 떠오르려 하지만 재질 식별 예외로 그대로 가라앉아 눌러앉는다(광석 옆에 있어야 읽혀서 플럭스로 작동하므로) — 그 아래 완성품 **Molten Metal**에 파묻히면 그 예외가 풀려 Ash/Sawdust와 같은 "가벼운 가루" 로직으로 위로 떠올라 그 층 밖으로 빠져나온다. 이 3종 밖의 일반 액체(물·기름·용암·수은·우라늄 용액 등)에서는 재질 예외 없이 다른 가루와 똑같이 밀도(5)로 뜨고 가라앉는다 — Mercury·Molten Uranium(9·10)에는 뜨고, 그보다 가벼운 액체에는 가라앉는다. |

## 🛢️ 석유 (Petroleum)

원유(Crude Oil)를 **끓여서 분별증류**하는 계열. 핵심 사용법: **Stone/Iron으로 용기를
만들고 그 아래에서 일반 불(Fire)/용암으로 가열**하면, 벽을 타고 전도된 열이 원유를
데워 끓는점 순으로 유분이 갈라져 나온다. 원유·유분은 **불꽃이 직접 닿으면 잘 탄다**(정상
가연성). 다만 **간접적인 열만으로는 발화하지 않는다** — 불꽃이 없으면 원유는 발화점에
닿기 전에 아스팔트로 크래킹되고 유분은 재비등해 날아가 버리기 때문이다. 그래서 밀폐 용기를
아무리 뜨겁게 데워도 타지 않고 증류된다("불 직접 대면 연소, 벽 너머로 데우면 분별증류").

원유 셀은 온도대에 따라 **제자리에서 해당 유분 증기로 끓어** 위로 올라간다(묻힌 원유도
증기가 액체 사이로 기포처럼 상승하므로 배출구가 필요 없다): `150–168° LPG,
168–250° 휘발유, 250–312° 등유, 312–380° 경유`, 380° 이상이면 **아스팔트**(고체 타르)로
크래킹된다. 아래에서 가열하면 뜨거운 바닥→차가운 위의 온도 구배가 생겨 **높이별로
자동 분별**된다(무거운 유분·잔사유는 낮고 뜨겁게, 가벼운 유분·가스는 높고 차갑게). 증기는
올라가 응결해 액체가 되고, 응결된 유분이 다시 뜨거운 바닥으로 떨어지면 **재비등(reflux)**해
증기로 되올라가므로 스스로 순환한다. 액체 유분은 밀도가 달라 저절로 층을 이룬다(가벼운
것이 위): **휘발유(2.2) < 등유(2.35) < 경유(2.45) < 원유(2.6) < 물(3)**.

> **유분 비중(고증 튜닝):** 끓어오름 확률이 틱당으로 누적되므로, 폭이 넓은 첫 번째
> 대역에서 거의 모든 셀이 끓어버려 아래 분획이 거의 나오지 않는다(가스대 18° 폭에
> 8%/틱이면 ~77%, 휘발유대 82° 폭이면 ~99.9%가 끓어버림). 그래서 **각 분획대의 폭에
> 반비례하게 틱당 확률을 따로 설정**해, 모든 액체 대역에서 통과 셀의 **~39%**가 해당
> 분획으로 끓어오르도록 튜닝했다(폭×확률 ≈ 0.49): 가스대 0.3%/틱(~5%), 휘발유 0.6%/틱
> (~39%), 등유 0.8%/틱(~39%), 경유 0.7%/틱(~38%). 결과적으로 **휘발유가 가장 많고**
> 등유·경유·아스팔트도 충분히 관찰된다(대략 가스 5% / 휘발유 37% / 등유 23% / 경유
> 13% / 아스팔트 22%). 또 **증기 응결을 빠르게** 해(응결 확률 상향) 증기가 큰 구름으로
> 남지 않고 금세 액체 유분으로 비처럼 떨어진다.

> 히트 브러시(+)로도 데울 수 있지만 온도 상승이 빨라 조절이 어렵다 — **Stone/Iron 용기 +
> 불 가열**이 열전도로 천천히 데워져 증류를 관찰하기 좋다.

> **렌더링:** 석유 계열 액체(원유·휘발유·등유·경유)는 배경 틴트를 샘플링하지 않고
> **단색으로 렌더링**되어 한 덩어리의 균일한 유체로 보인다(나머지 액체는 미세한 색 변화).
>
> **유류화재:** 물 위에 석유를 붓고 점화하면 **표면에서 계속 타는 기름불**이 된다 — 물이
> 불을 끄지도, 아래 물이 빠르게 증발하지도 않는다(물과 닿은 석유층은 소화·소진되지 않고
> 물을 가려 보호).

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Crude Oil | 2.6 | 원료. 간접(전도) 가열 시 분별증류, 불꽃 직접 접촉 시 연소(두 번째로 빨리 탐). |
| LPG | 0.8(기체) | 가장 먼저 끓어 나오는 경질 가스 산물. 상승·확산. 폭굉 없이 **폭연(연소)만** 하는 가연성 기체 — 불꽃 접촉/400°↑ 자연발화 시 셀 단위로 화염 전파(충격파·크레이터 없음). |
| Petroleum Vapor | 1.2(기체) | 증류 중간 증기. 상승 후 응결해 해당 유분(휘발유/등유/경유)이 됨. |
| Gasoline | 2.2 | 가장 가벼운 액체 유분. 가장 빨리 타는 연료. 200°↑ 재비등(reflux). |

| Diesel | 2.45 | 무거운 액체 유분(경유). 연소 속도를 Resin 수준으로 낮춰 전체 연료 중 가장 느리게 탐(Resin과 동률). 320°↑ 재비등. |
| Asphalt | (고체/고온 액체) | 잔사유(타르). 원유가 380°↑에서 크래킹된 잔재. 식으면 고체 크러스트, **고온(200°↑)이면 점성 높은 액체로 흘러내림**. 비가연성. |

| 물질 | 거동 / 상호작용 |

| Ice | 고체 얼음. 녹으면 Water. |
| Snow | 가벼운 가루 눈(밀도 2). 흩날리듯 좌우로 흔들리며 천천히 떨어진다. 녹으면 Water, 더 얼면 Ice. 물에 파묻혀도 스스로 수면 위로 떠오른다(Ash와 동일한 부력). |

Sand·Salt(5)보다 가벼운 유기 결정이라 물속에서 확실히 잠긴 채 젖어 냉각 반응을 계속한다. **ANFO:** Diesel/Kerosene을 부어 겹침(overlap)으로 스며들게 하면(물은 스며들지 않고 그대로 겹침 반응·젖음 판정용 primary 셀로 남음) 이후 기폭 시 뭉친 양과 무관하게 고정된 **TNT 80% 위력**으로 폭발 — 실제 ANFO처럼 연료유를 먹인 쪽이 훨씬 강력하다. |

## ⚡ 전기 (Electric)

> **전기 세기 시스템:** 스파크는 이동하며 **세기가 감쇠**한다 — 금속(Iron·Mercury·Nichrome)에서는
> 온전히 유지(니크롬의 저항은 세기 감쇠가 아니라 줄열로만 나타난다), 소금물에서는 천천히,
> 물에서는 빠르게 약해져 몇 칸 못 가 소멸. 도체(금속·수은·니크롬·물·소금물)만 전기를
> 통하고 **그 외 물질은 절연체**로 완전 차단한다. 전기전도는 더
> 이상 연료·가연성 기체를 **점화하지 않지만**(인화성 제거) **폭약(TNT 등)은 여전히 기폭**한다.
> 물·소금물에 전기가 흐르면 **전기분해로 소량의 수소·산소를 생성**한다(낮은 확률).

| 물질 | 거동 / 상호작용 |
|---|---|
| Iron | 최고 열전도 + **전기 도체**(세기 유지). 1200°↑ 용융(맨 석탄불(1300°)만으로 도달 — 권장 화력). |
| Nichrome | **전열선(저항 발열체)**. 전기 도체로서의 저항은 엔진 최저치(금속과 동일 — 세기 감쇠 없이 전선 끝까지 온전히 도달). **전기가 지나갈 때마다 저항열(줄열)로 가열**되어 회색 합금이 벌겋게 달아오름 — 맨 선은 공기 중 **복사 방열**과 평형을 이뤄 ~750° 백열(전원 끊으면 몇 초 만에 식음), 물·모래 등에 **묻으면 방열이 막혀 최대 1050°까지** 오르며 그 이상 자가 가열은 포화해 **스스로는 절대 안 녹는다**(전열선의 본분). 실물처럼 **양 끝 단자는 발열하지 않는 콜드 엔드**(3칸 이상 깔아야 가운데가 달아오름)라 배선으로 열이 덜 새어나간다. 열전도율도 낮아(실제 니크롬처럼 철의 1/6) 줄열이 코일에 머문다. 300°↑면 인접 가연물을 무화염 점화. **내산성**(산 속 배선 가능 — Iron 은 녹는다). 외부 열원으로 1450°↑ 시 용융. |
| Lithium Battery | 주기적으로 인접 도체에 Spark 주입(full 세기) → 자가 구동 회로. 4×5 타일에 대각 2계단 픽셀 아트 배터리 패턴 적용. **고온(200°↑)이면 열폭주 — 잠깐 불붙어 타다가 폭발**(일반 화염 열로도 기폭). 평상시엔 **완만한 자체 방열**(방열판/보호회로)이 있어 배선을 타고 오는 은근한 열(예: 니크롬 줄열의 역류)은 버티지만, 직화에는 여전히 순식간에 폭주한다. |
| LFP Battery | 안전한 배터리 화학 — 리튬 배터리와 **동일한 주기·세기로 전기 공급**하며 4×5 타일에 대각 2계단 픽셀 아트 배터리 패턴 적용, **연소·폭발(열폭주)이 전혀 없다**. 대신 방폭은 아니어서 폭발 피해에는 일반 고체처럼 파괴되고, Blue Flame·Lava·Thermite 급 극고온(1400°↑ — 철(1200°)보다 높게 잡아 석탄불엔 안 녹음)에서는 녹아내린다(→Molten Metal). 화염 속·니크롬 히터 옆에 둘 전원. |
| Woofer | **비파괴 충격파 가전** — 전기가 닿을 때마다 **물리적으로는 투명한**(섬광·잔불 없는) 충격파를 발산해 주변 가루·물을 강하게 밀어낸다(폭발 파괴력/내구력 축을 0으로 고정 재사용 — 어떤 고체도 못 부숨, 완전 비파괴, 크레이터 섬광도 없어 인접 배터리를 잔불로 폭주시키지 않고 화약·TNT 등도 오작동하지 않음). **충격파 이펙트** — 발동 시 **몸체 외곽선에서 바깥으로 퍼지는 시안색 픽셀 파동**을 Fan의 바람처럼 **배경 효과로 렌더링**한다(단일 시안 레이어 + **퍼져 나갔다 사라지는 얇은 파면** + 베이어 디더 도트 느낌). **범위 정직** — 파면이 **몸체 셀을 시드로 한 측지 거리장**에서 자라 캐비닛의 실제 모양대로 실제 미는 거리만큼 퍼지므로(중심의 원이 아님), 물질이 밀리는 영역을 그대로 덮고 **고체 벽은 파동을 막는다**(거리장이 고체를 통과 못 함 — POWER 0 펄스가 고체에 막히는 물리와 일치). **물질이 파동을 가리되, 액체만 체커보드 스티플로 반투명 처리**해 파동·배경이 웅덩이 너머로 비쳐 보인다(오직 Woofer 충격파에만). 물리는 여전히 투명하고, 링은 순수 코스메틱(시뮬 결정성 무관). 폭약이 아니라 **소모되지 않는 상시 설치물**이라 펄스마다 반복 발동 가능. **자체 전도성**(도체 태그 없이 배터리·터빈·도체 릴레이 모두에서 직접 인식) — **배터리나 터빈에 배선(iron 등 도체) 없이 직접 붙여도 작동**하고, 연결된 우퍼 몸체 전체가 즉시 동시 발동하며(터빈의 내부→외부 방출과 반대로 외부→내부만 허용), 내부에 Spark가 렌더링되지 않는다. **독립 오브젝트**(공·드럼통·다이너마이트)도 전용 이벤트 큐로 밀어내지만 절대 파괴하지 않는다(OBJECTS.md 참조). |
| Fan (선풍기) | **바람을 일으키는 전기 가전**. 컨베이어처럼 **배치할 때 드래그한 방향으로 바람 방향**이 정해진다(상하좌우 4방향 — 셰브런으로 표시, 작동 중엔 밝아짐). 전기가 닿으면 앞쪽으로 최대 **45칸의 바람**을 뿜어 범위 내 **가루·액체·기체·독립 오브젝트(드럼·공 등)를 그 방향으로 밀어낸다**(기체 구름도 제대로 밀려 흘러가고, 밀린 기체는 **옆으로 부채꼴로 퍼진다** — 위쪽 바람에도 한 줄로 안 뭉치고 확산). **밀어내기 세기 1.5배** — 밀린 물질이 평균 ≈1.5칸/틱으로 더 세게 흐른다. **고정 고체(벽·돌·기계·회로 등)는 안 밀리고 바람을 막는다** — 자기 회로·구조물이 날아가지 않게(고체 물체는 오브젝트 레이어에서 밀림). **대류 냉각** — 바람 범위 내에서 **상온(20°C)보다 뜨거운 물질**(빔이 막힌 고체 표면 포함)은 상온 쪽으로 **천천히 식는다**(상온 아래로는 안 내려감). 바람은 **파티클이 아니라 임시 바람 필드**(Grid.wind)로, 매 틱 갱신되고 선풍기가 멈추면 즉시 사라진다. **단방향** — 물질에 힘만 주고(밀기) 물질에 막히거나 밀리지 않으며, 렌더러가 그 필드 위에 **흐르는 저해상도 바람 줄무늬 애니메이션**을 배경 효과로 그린다. 바람은 **임시 효과**라 선풍기가 유지하지 않으면 즉시 사라진다. **자체 전도성 없음**(Woofer와 같은 외부→내부 단방향 싱크) — 배터리·터빈에 직접 붙이거나(도체 없이) 배선으로 연결하면, 전기가 닿는 순간 **연결된 선풍기 몸체 전체가 동시에 켜진다**. 배터리 펄스는 주기적이지만 **점화 지속시간을 넉넉히 잡아 바람이 깜빡이지 않는다**. |
| Laser (레이저) | **열선(Heat Ray)을 쏘는 전기 방출기**. Fan과 똑같이 **배치할 때 드래그한 방향으로 발사 방향**이 정해진다(상하좌우 4방향 — 셰브런으로 표시, 작동 중엔 밝아짐). **자체 전도성 없음**(Fan/Woofer와 같은 외부→내부 단방향 싱크) — 배터리·터빈에 직접 붙이거나 배선으로 연결하면 전기가 닿는 순간 **연결된 몸체 전체가 동시에 켜진다**. 엔진 전기는 **주기적 펄스**라 한 틱만 살면 깜빡이므로 Fan처럼 **점화 지속시간을 넉넉히**(24틱) 잡아 펄스 간격을 메워 **끊김 없이 발사**한다. 점화 중엔 매 틱 **총구(발사 방향 바로 앞 빈 셀)에서 열선 한 발**을 쏴 연속 레이저처럼 읽힌다. **내산성**(다른 전기 기계처럼 산·부식으로 안 사라짐). 발사되는 열선은 **파괴력 없이 가열만** 하고, **유리 투과·반사성 금속 정반사·Diamond 프리즘 확산·기체 산란·액체 관통** 특성을 가진다(위 팔레트 비노출 시스템 파티클 열선 참조). |
| (Heat Ray) | Laser가 전기를 받아 쏘는 **레이저 빔**(핵 광선과 같은 8방향 직진 비행이되 **파괴력 0** — 셀을 절대 지우지 않고 오직 가열). **막는 고체·가루**는 명중 셀 +140°/인접 +45°로 지진 뒤 **대부분(≈82%) 그 자리에서 소멸(가열 후 소멸)**, **18%만 난반사**로 튕김. **유리·깨진유리 투과**. **반사성 금속은 정반사**(거울) — Mercury·Iron·Heatpipe·Gallium·Liquid Gallium(모두 `laserReflective` 플래그, 이후 반사율 높은 금속은 플래그만 달면 자동 적용). 평평한 면은 되돌리고 **45° 대각선 금속 면은 빔을 90° 꺾음**(대각선 정반사 — 계단식 대각선으로 레이저를 모퉁이로 보냄). **Diamond는 프리즘** — 진입은 그대로, **빠져나오는 지점에서 사방으로 확산**(스타버스트). **기체 통과 시 6% 확률로 45° 산란**. **액체는 대부분 관통** — 지나며 낮은 확률로 **흡수·소멸(6%, 소멸 지점 +130°)** 또는 **산란(5%)**, 나머지는 그대로 통과. 열지도에선 `overlayTemp`로 백열 고정 표시. 오브젝트는 파괴하지 않고 지나간다. |
| (Spark) | 도체를 타고 전파하는 전기 펄스(매질별 세기 감쇠). 폭약 기폭(연료는 점화 안 함), Woofer 충격파 발동(단, Woofer 자체에는 Spark가 나타나지 않음). 불응 처리로 역류 방지. **배선 없이 배터리·터빈에 직접 맞닿은 폭약도 동일하게 기폭**(전원 3종·스파크 공용 디스패치 `reactToPulse` — Woofer·Fan 가전과 같은 경로). |

## 🌱 생명 (Life)

| 물질 | 거동 / 상호작용 |
|---|---|
| Plant | **물을 빨아올려 위로 자라는** 식물. 뿌리(수면)가 물을 마시면 그 수분이 줄기를 타고 올라가고, 끝에서 빈칸(위쪽 우선)으로 가지치며 성장 — 물이 닿는 만큼 높이 오르다 물이 마르면 멈춘다. 마른 초목은 불이 잘 붙어 번짐(젖으면 소화). |
| Virus | 유기물·흙·물을 감염시켜 전파. 불·산·열로 치료. **H2O2**가 닿으면 랜덤 방향으로 부식되며 소멸(H2O2 양에 비례 — 무한 확산 없음), **Alcohol은 닿은 부분만** 살균. |
| Yeast | 효모(가루, 밀도 3.3). 설탕(Sugar)/꿀(Honey)과 물 사이에 있으면 **발효**: 물→Alcohol, 당→CO₂ 거품(효모는 촉매로 남아 계속 변환). 60°↑ 열에 죽음. 물보다 무거워 가라앉으므로 발효 대상과의 접촉을 유지한다. |
| Seed | 씨앗(가루, 밀도 3.45). 젖은 흙·진흙 위에 앉으면 시간이 지나 **발아→Plant 새싹**(물 있는 만큼 위로 성장). 마르거나 흙에서 떨어지면 발아 정지. 물보다 무거워 가라앉으므로 침수된 밭에서도 바닥 흙까지 내려가 발아할 수 있다. |
| Slime | 초점성 녹색 반유체. 무거운 goo라 **낙하가 느림**(물처럼 곧장 떨어지지 않고 천천히 가라앉음). 천천히 흐르며 인접 물을 흡수해 **몸집을 키운다**. 불꽃/열엔 **녹아 연기로 소멸**. **전기전도성**이 있으며 물과 동급 수준으로 상향(~7칸 도달) — 관통한 전류가 낮은 확률로 **전기 분해 부식**을 일으켜 물로 되돌리는데, 한 번의 부식이 닿는 범위(예산)도 바이러스의 H₂O₂ 부식보다 2배로 넓어 지속 전류 시 blob이 더 빠르게 무너진다(지속 전류 필요). 전기로 분해된 물은 잠시 동안 다시 흡수되지 않아(재흡수 유예) 분해수가 blob에서 빠져나갈 시간을 번다. |
| Acid Slime | Slime의 산성 사촌. **거의 동일한 슬라임 매커니즘**(초점성 오즈·**느린 낙하**·인접 물 흡수로 성장·불꽃/열에 녹아 연기로 소멸)에 **Acid와 동일한 부식력**을 더함 — 매 틱 인접 비내산성 고체·가루를 부식(동일 CORRODE_CHANCE), 부식 시 자기 소모로 소멸 가능(무한 부식 방지). 일반 Slime과는 부식하지 않고 **단순 상호확산으로 섞임**(물+Acid처럼, 같은 밀도라 경계에서 서서히 혼합). **전기전도성 최대치**(로스터에서 유일하게 금속과 동급인 무손실 전도체 — 펄스가 blob 전체를 관통). 전기 약점은 Slime과 **동일** — 관통한 전류가 낮은 확률로 **전기분해 부식 front**를 만들어 물로 되돌리며 이웃 Acid Slime으로 프레이 확산(지속 전류 시 blob이 무너짐). 전기로 분해된 물은 잠시 동안 다시 흡수되지 않아(재흡수 유예) 분해수가 빠져나갈 시간을 번다. |
| Termite (흰개미) | 고체·가루 표면을 **기어다니는** 벌레(바닥·벽·천장·월드 경계 표면을 훑는 우변 벽타기, 이동방향 유지). **액체는 기피** — 수면·물가를 따라 걷되 들어가지 않음. 틱당 5% 확률로 인접한 **Sawdust·Wood를 갉아먹어 흰개미로 번식**. 사망 시 **Sawdust**로 남음(= 먹이라 군집이 자급). 죽는 조건: **물에 완전히 잠김**, **폭발 충격파(단 Woofer 제외)**, **70°↑ 열**. 이동방식·죽음 판정은 Nanobot과 공유(`crawler.ts`). |
| Nanobot (나노봇) | 금속을 먹는 기계 벌레 — 흰개미와 **동일한 기어다니기**. **액체는 무시**하고 통과해 물속 금속 표면까지 기어다님. 틱당 5% 확률로 인접한 **Iron·Metal Powder를 갉아먹어 나노봇으로 번식**(Rust 계열 물질은 먹지 않고 무시). **Metal Powder와 같은 녹는점**(Iron 융점)에서 **Molten Metal로 융해**, **폭발 충격파(단 Woofer 제외)** 노출 시 **Metal Powder로 파괴**, **소금물(Saltwater) 노출 시 Rust(녹)로 부식**. |

## ☢️ 방사성 (Radioactive)

우라늄 계열 전용 카테고리. **모든 우라늄 계열 물질은 방폭**(폭발 면역) — Blast 크레이터·충격파 압력전파·불씨·반물질 모두 무시한다(고체·액체·가루 상 무관). **단 Nuclear Ray 반응은 예외**로, 방폭이어도 임계 우라늄의 핵 광선에는 기존과 동일하게 반응한다(고체 우라늄은 핵 광선 명중 시 멜트다운).

| 물질 | 거동 / 상호작용 |
|---|---|
| U235 | 방사성 고체. 인접 우라늄(U235/Molten U235) 수만큼 자가 발열(연쇄반응) — 소량은 안전, 두꺼운 덩어리는 폭주. 물 냉각으로 원자로 운전 가능. 1500°↑ → Molten U235 로 **멜트다운**. **방폭.** |
| Molten U235 | 녹은 우라늄(코륨). 점성 없이 물처럼 흐르고 **모든 액체 중 최고 밀도(10)**. 발열은 더 빨라져 계속 가열되며, 1400°↓ 냉각 시 다시 고체 U235 로 응고. **2000°(임계) 도달 시 표면에서 서서히 연소하며 Nuclear Ray 방출** — 즉발 폭발이 아니라 몇 초에 걸쳐 타들어간다. **덩어리로 크게 뭉치면 표면뿐 아니라 내부 셀까지 빠르게 붕괴**해 대량 폭주가 오래 끌지 않는다. **방폭.** |
| U238 | 또 하나의 방사성 고체. U235처럼 인접 우라늄 수만큼 자가 발열하고 물 냉각으로 운전 가능하지만, **비폭발성 — 멜트다운까지만** 간다. 1500°↑ → Molten U238. 임계·폭발·Nuclear Ray 없이 그냥 녹아내리는 "느리고 지저분한" 연료. **방폭.** |
| Molten U238 | 녹은 U238. **멜트다운과 동시에 연쇄반응이 멈춘다** — 자가 발열 없이 서서히 식기만 한다(공기 중 방사 냉각 + 물 냉각 가속). 1400°↓로 식으면 **되돌아오지 않고 Nuke Waste(가루)로 굳는다** — U235의 가역 응고와 달리 **일방통행**. **방폭.** |
| Nuke Waste | U238 연료 사이클의 최종 잔재(핵폐기물) 가루. **연쇄반응이 남아있지 않아 다시 멜트다운하지 않는다.** 붕괴열로 **약한 열원** 역할(스스로 ~160°까지 데워 주변에 은은한 잔열 전달, 무언가를 점화할 만큼 뜨겁진 않음). **방폭.** 쌓고 묻고 버리는 용도. |
| (Nuclear Ray) | 임계 우라늄이 뿜는 **초고온 붉은 광선**. 8방향 직진, Wall·경계에서 **랜덤 각도로 반사**(규칙적 궤적 없이 카오스하게 확산), 서로는 간섭 없이 투과. 우라늄은 파괴하지 않는다 — **고체 우라늄(U235·U238)은 명중 즉시 멜트다운**(→각각 Molten U235/U238)하고, 녹은 우라늄은 +800° 대량 가열; Molten U235는 그 셀의 **붕괴 촉발**(가열+붕괴 가속)로 임계 풀을 광선이 스스로 갉아먹지만, 비폭발성 Molten U238은 냉각만 지연될 뿐 붕괴 가속은 없다. 스치는 우라늄 침전물을 즉시 액화시켜 연쇄 점화가 파일 사이로 빠르게 번진다. 그 외 모든 물질(Diamond·물 포함)은 파괴 후 확률적으로 관통 직진 또는 반사하며, 충돌 지점 주변을 강하게 지져(+400°) 녹이고·끓이고·태운다. **반사할 때마다 수명을 더 소모**하므로 지형을 관통하며 날아가는 광선은 화면을 가로지르지만 구석·발생원에 갇혀 반복 반사하는 광선은 곧 소멸(뭉침 방지). **또한 대량 붕괴로 광선이 빽빽이 쏟아질 때, 온통 형제 광선뿐이라 빈칸에 착지 못하고 제자리에 갇힌(형제끼리는 투과만 가능) 광선도 반사처럼 수명을 크게 소모**시켜 뭉침(클럼프)을 빠르게 해소한다. 우라늄이 충분하면 화면 전체를 초기화 수준으로 밀어버린다. **독립 오브젝트(공·드럼통·다이너마이트)를 스치면 Blast 직격과 똑같이 즉시 파괴**한다(docs/OBJECTS.md 참고). 열지도 오버레이에서는 항상 백열색으로 고정 표시돼(`overlayTemp`) 다른 패킹값 재료처럼 배경에 묻히지 않는다. |

## 대표 상호작용 사슬

- **금속 주조:** Iron → (열) → Molten Metal → (냉각) → Iron. Water 부으면 Steam 으로 냉각. 철은 1200°에서 녹으므로 **맨 석탄불(1300°)**만으로 충분하다 — 이 온도는 돌(1100°)도 녹이니 노·용기는 안 녹는 **Diamond·Heatpipe**로 짓는다.
- **제련(1단계 용해):** Iron Ore 를 일반 Fire(또는 Lava·Blue Flame·석탄불)로 850°↑까지 데우면 **Molten Iron Ore**(용융 웅덩이)로 녹는다. 녹이기만 하고 두면 식어서 Slag(실패).
- **제련(2단계 환원·주조):** 용융 웅덩이에 **Coal Powder 를 뿌리면**(가라앉아 섞임) 또는 광석과 미리 섞어/층으로 쌓아 가열하면(석탄이 타지 않고 환원) → 발열 반응으로 뜨거운 Molten Metal 이 되어 바닥에 고이고, 식으면 Iron. 슬래그(위)와 쇳물(아래)이 층으로 갈린다. Limestone 을 더하면 철 수율↑.
- **유리·소금 용융:** Sand → Molten Glass → Glass, Salt → Molten Salt → Salt (같은 상전이 패턴).
- **전기 기폭:** Lithium Battery → Iron/Mercury 배선 → Spark(세기 감쇠) → 아크 불꽃 → Gunpowder/TNT 기폭(연료는 점화 안 함).
- **전기분해:** 물·소금물에 전기가 흐르면 소량의 수소·산소 생성(수소는 산소와 만나 물로).
- **전기 곤로/주전자:** Battery → Iron 배선 → **Nichrome 코일**: 전기가 흐를 때마다 줄열로 달아올라(글로우) 물을 끓이고(수조 바닥 코일 = 전기 주전자) 소금을 녹이고 기름·나무를 무화염 점화 — 불 없는 전기 히터. 물이 다 끓어 마르면(boil-dry) 코일이 1000°+ 까지 치솟으니 리튬 배터리 배선을 짧게 붙이면 위험하다 — 극한 환경 전원은 열폭주 없는 **LFP Battery**로.
- **소화·중화:** Soda 를 화재에 뿌리면 불꽃 소화 + 타는 연료 냉각 진화(가루 소화기), Acid 웅덩이에 부으면 거품을 내며 Saltwater 로 중화.
- **도화선 폭파:** Fire → Fuse(지연) → TNT 연쇄.
- **냉각:** Dry Ice(승화해 **차가운 CO₂ 안개**를 뿜음) / LN₂(-196°) 가 물을 얼리고 불을 끔.
- **질식 소화:** CO₂(무거운 기체)가 바닥에 깔려 불을 덮어 끄고 **타는 연료까지 식혀** 재발화를 막는다(광역 진화) — Dry Ice 를 태우는 불 옆에 두면 CO₂ 안개로 자가 소화.
- **절단:** Thermite 점화 → 돌·철·유리를 뚫으며 녹임(붙어도 계속 낙하). 절단 후엔 뜨거운 용융 잔재가 식으며 잔류물 없이 서서히 소멸.
- **연소·제조수:** 불 → Hydrogen(가연성 기체) 연소, Oxygen 과 만나면 물(수증기) 생성 — 폭발이 아니라 조용한 연소.
- **유류화재:** 물 위에 석유(원유·휘발유 등)를 붓고 점화 → 표면에서 계속 타는 기름불, 아래 물은 증발 억제.
- **불멸 구조:** Diamond 는 모든 폭발·고온·저온·산에 면역이면서 열은 전도.
- **생태계·재배:** Water + Dirt → Mud, **Seed 를 젖은 흙에 심으면 발아 → Plant** 가 물을 빨아올려 위로 자람(물이 마르면 멈춤), Virus 가 유기물을 감염.
- **발효:** Yeast + (Sugar/Honey) + Water → **Alcohol + CO₂** — 만들어진 알코올은 그대로 연료 사슬로 이어짐.
- **살균:** **H2O2**가 Virus 에 닿으면 부식이 랜덤 방향으로 우둘투둘 번져 소멸시키되, 살균할 때마다 H2O2가 소모돼 **뿌린 양에 비례**한 만큼만 지운다(극소량으론 대량 군집을 다 못 지움 — 무한 확산 없음). **Alcohol**은 닿은 셀만 없앰(확산 없음).
- **나트륨 반응:** Sodium 을 물에 던지면 불꽃+수소 화구, **뭉쳐 쌓으면 물 접촉 시 덩어리째 폭발**(양에 비례).
- **자연발화:** White Phosphorus 는 공기에 노출되면 스스로 불붙어 백연을 뿜고, 물에 담그면 꺼진다.
- **경화 ↔ 용해(가역):** Resin 이 시간이 지나 굳어 Amber(호박)가 됨(400° 미만). 반대로 **불타는 Amber**(700°↑)는 낮은 확률로 **일부만** 다시 Resin 방울로 녹아 흐른다 — 덩어리 전체가 녹지 않고 대략 1/10 정도만 끈적하게 흘러내리고 나머지는 나무처럼 타 재가 되는, "가끔 수지가 배어나오는" 화석 수지 연소. Aerogel 로 열을 완전히 차단한 실험 격벽.
- **소스/싱크:** Clone(무한 생성) ↔ Void(무한 삭제) 컨베이어.
- **원자로/멜트다운(U235):** U235 덩어리 자가 발열 → 물 냉각(Steam 배출)으로 운전 ↔ 냉각 중단 시 1500° 멜트다운 → Molten U235(물처럼 흐르는 최고 밀도 액체) → 재냉각하면 고체 복귀, 방치하면 2000° 임계 → Nuclear Ray 방사로 화면 전체 소거.
- **U238 사이클(비폭발):** U238 덩어리 자가 발열 → 1500° 멜트다운 → Molten U238(연쇄반응 정지, 자가 발열 없음) → 서서히 식으며(방사 냉각 + 물 냉각) 1400°↓에서 **Nuke Waste(가루)로 일방통행 응고** → Nuke Waste는 약한 잔열원. 임계·폭발 없이 "폐기물 더미로 식어버리는" 결말.
- **방폭:** 모든 우라늄 계열(U235·Molten U235·U238·Molten U238·Nuke Waste)은 폭발에 완전 면역 — 옆에서 TNT가 터져도 크레이터·충격파에 파괴/산란되지 않는다(단 Nuclear Ray에는 기존대로 반응).
