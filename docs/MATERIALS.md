# 물질 도감 (Materials)

전체 물질과 핵심 상호작용 요약. 팔레트는 아래 카테고리 탭으로 묶인다(각 물질의
`category` 필드, 생략 시 `phase` 에서 자동 유도).

> `Eraser`(지우개), `Ember`(불씨), `Spark`(스파크), `Heat Ray`(열선)는 팔레트에
> 없다: 폭발 파편/전기 펄스/임계 열선으로 시뮬레이션이 스스로 생성하는 일시적
> 입자다.

## 🪨 고체 (Solid)

| 물질 | 거동 / 상호작용 |
|---|---|
| Wall | 파괴 불가 경계. 폭발·산·Void 모두 못 뚫음. **온도 시스템 완전 배제**(자체 온도 없음, 열전도 0 — 완벽 단열 경계). |
| Stone | 정적 장벽. 1100°↑ → Lava 로 용융. |
| Glass | 투명. 내산성. 1250°↑ → Molten Glass 로 재용융. |
| Concrete | Cement + 물이 굳은 결과물. 강하지만 폭발·산에 취약. |
| Diamond | 최고 열전도. **모든 폭발·고온·저온·산에 면역**. 방폭·전열 구조물. 유일한 예외: 우라늄 임계 열선(Heat Ray)은 다이아몬드도 뚫는다. |
| Amber | 송진(Resin)이 시간이 지나 굳어 만들어지는 반투명 황금빛 고체. 장식·봉인용("호박 속 곤충" 밈). 여전히 나무 수지라 **가연성**(느리게 탐). |
| Aerogel | **완전 단열재**(열전도 0) — 열이 전혀 안 통해 용암/얼음 실험 격벽으로 최적. 단, **방폭 아님**: 열만 막고 힘은 못 막아 폭발엔 일반 고체처럼 부서지고 산·테르밋에도 당한다. |

## 🏖️ 가루 (Powder)

| 물질 | 거동 / 상호작용 |
|---|---|
| Sand | 낙하·퇴적. 1250°↑ → Molten Glass. |
| Salt | 물에 용해 → Saltwater. 초고온(800°↑) → 용융염(Molten Salt). |
| Soda | 베이킹소다 — **분말 소화기 + 산 중화제**. 인접한 불을 즉시 소화하고 **타고 있는 연료도 식혀서 진화**(소화 작업마다 조금씩 소모 — 김을 내며 사라짐). Acid 접촉 → **Saltwater 로 중화**(소금+물, 실제 반응 산물) + 거품. Acid Vapor 도 흡착 제거. 150°↑ 가열 시 분해되어 김으로 소멸. |
| Dirt | 물 흡수 → Mud. 생태계의 지반 층. |
| Cement | 물 접촉 → Concrete 로 경화(물 소모). |
| Ash | 연소 잔재. 아주 가벼워 물에 뜬다. 가루가 흩날리듯 좌우로 흔들리며 천천히 떨어진다. |
| Sodium | **물에 닿으면 격렬 반응**: 불꽃+수소 방출+발열. 소량은 물 위를 스치며 타는 화구, **뭉쳐 쌓으면 물 접촉 시 덩어리째 폭발**(양에 비례). 물보다 가벼워 수면에서 반응. |
| Sugar | 설탕. 불에 잘 타는 가연성 가루. **가열(200°↑)하면 캐러멜화→탄화(→재)**. 물엔 안 녹음(효모용) — **효모+물과 만나면 발효**(→Alcohol+CO₂). |

## 💧 액체 (Liquid)

> **어는점(경량 방식):** 모든 일반 액체는 각자의 어는점 이하로 냉각되면 **제자리에서 굳어**
> — 흐르지 않고 고체처럼 밀도 치환도 막으며 서리색으로 표시 — 데우면 다시 흐른다. 별도
> 물질로 바뀌지 않는다(물만 예외 — Snow/Ice 로 상전이). 물·소금물은 **전기 도체**이기도 하다.

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Water | 3 | 100° 끓음→Steam, 0°↓ 얼음/눈. **전기 도체**(약함 — 스파크가 몇 칸 못 가 감쇠 소멸). 전기 통하면 소량의 수소·산소 생성(전기분해). |
| Saltwater | 4 | 물보다 무겁게 가라앉음. 끓으면 소금 석출. -18°↓ 동결. **전기 도체**(약전해질 — 스파크가 물보다 멀리 전달). 전기분해로 소량 수소·산소. |
| Acid | 3 | 비내산성 고체·가루 부식. 끓으면 Acid Vapor. -20°↓ 동결. |
| Mercury | 9 | 가장 무거운 일반 액체(Molten Uranium(10) 다음). **전기 도체**(스파크 전달). -38°↓ 동결. 단색 렌더(배경 틴트 샘플링 제외). |
| Honey | 3.5 | 점성 높아 천천히 흐름. 느리게 타는 연료. **물과 섞임**(Acid처럼 상호확산). 5°↓ 굳음(결정화). |
| Alcohol | 1.9 | 가장 가벼운 연료. 순식간에 옮겨붙어 확 탄다. -80°↓ 동결. |
| Mud | 5.5 | 젖은 흙. 마르면 Dirt 로 굳음. -3°↓ 동결. |
| H2O2 | 3.4 | 과산화수소. 스스로 분해하며 **산소 방출**(→불 가속제)+물로 환원. 가열·촉매 시 분해 급가속(불 근처면 산소 폭주). **바이러스 살균**. |
| Resin | 2.6 | 송진(물에 뜸). 끈적한 가연성 액체. 시간이 지나면 **굳어 Amber(호박)로 경화**. 데우면 굳기 전에 탄다. |

## 💨 기체 (Gas)

| 물질 | 거동 / 상호작용 |
|---|---|
| Steam | 상승 후 응결 → Water. |
| Smoke | 상승 후 확률 소멸(수명 짧음 — 예전의 1/3, ~0.6초). |
| Acid Vapor | 상승하며 부식, 응결 → Acid. |
| Oxygen | 불 접촉 시 자신도 불로 확 번짐(가속제). 수소와 함께 연소하면 수증기(→물) 생성. |
| Hydrogen | 가장 가벼운 **가연성 기체**(폭약 아님). 점화 시 연소해 불로 번짐. **Oxygen 과 만나 점화되면 물(수증기) 생성**(2H₂+O₂→2H₂O). |
| CO2 | **무거운 기체** — 바닥에 깔려 고이고 낮은 곳을 채운다(최초의 하강 기체). 불을 덮어 **질식 소화**. Dry Ice 승화물(차가운 CO₂ 안개). 서서히 공기로 소멸. |
| Chlorine | 무겁고 **독성** 기체 — 바닥에 깔려 생물(Plant·Virus·Yeast·Seed·Slime)을 살상. 서서히 소멸. (원래 표백제+산으로 생성 — 표백제 미구현이라 현재는 직접 배치) |
| Helium | **초경량** — 모든 기체 위로 상승해 천장에 고임. 무반응(풍선 개그). |

## 🔥 불·열 (Fire / Heat)

| 물질 | 거동 / 상호작용 |
|---|---|
| Fire | 가연물 점화, 물에 소화. |
| Blue Flame | 초고온. 돌을 녹여 Lava 로. **연기 미생성**(깨끗한 절단 토치). |
| Lava | 용융 암석. 식으면 Stone. |
| Molten Metal | Iron 이 녹은 것. 식으면 Iron. 밀도 8 액체. |
| Molten Glass | Sand 가 녹은 것. 식으면 Glass. |
| Molten Salt | Salt 가 녹은 것(800°↑). 식으면 Salt. |
| Coal / Wood / Sawdust | 고체 연료(느림→빠름 순: Coal < Wood < Sawdust). 액체 연료(원유·휘발유 등)는 아래 **석유** 탭 참고. |
| White Phosphorus | 백린(가루). **공기 중 자연발화** — 노출되면 백색 화염+짙은 백연(연막)을 뿜고 주변 가연물을 점화하며 계속 낙하·연소. **물속에선 안전**(잠기면 불활성, 담그면 소화). |

> **산소 송풍(forced draught):** 모든 연료는 연소 중 인접한 **Oxygen** 1칸당 화염 온도가
> +250°(상한 1550°) 오르고, 빨려든 산소는 확률적으로 소모된다. 맨 연료불은 800°지만
> 산소를 불어넣으면 1050°(1칸)·1300°(2칸)·1550°(3칸+)까지 올라 **철(1400°)까지 녹인다**.
> 연료 종류 무관이라 "산소+아무 연료 = 더 뜨거운 절단 토치"도 덤으로 성립하고, 산소를 문
> 석탄불은 아래 **제련** 탭의 철광석도 녹일 만큼 뜨겁다.

## ⚒️ 제련 (Smelting)

철광석에서 철을 뽑는 **제련 공정 놀이**. 정체성은 분별증류의 "직화=연소 / 간접열=증류"
대비를 잇는 **"그냥 녹이면 쓸모없는 슬래그(실패) / 탄소(석탄)와 함께 구우면 철(성공)"**
이다. 공정은 **2단계** — ① **철광석을 녹인다**(일반 Fire로도 충분, 850°↑), ② 녹은
**용융 철광석 웅덩이에 석탄가루를 뿌린다** → 철. 녹이기만 하고 석탄을 안 넣으면 용융
광석이 식어 **쓸모없는 Slag로 굳는다** — "열만으로는 슬래그, 석탄과 함께여야 철".

**1단계 용해:** 철광석(가루)은 850° 이상이면 **용융 철광석**(빛나는 액체)으로 녹는다.
Fire 셀은 ~1000°라 **불만 충분히 대면 녹지만**, 맨 석탄불(800° 핀)은 그 아래라 녹이지
못한다 — 그래서 "불로 녹이기"와 "석탄으로 환원"이 별개의 단계로 남는다(산소를 문 석탄불,
Lava·Blue Flame·히트 브러시도 녹인다). **2단계 환원:** 용융 웅덩이에 닿은 **석탄가루/
석탄**이 광석을 환원한다 — **석탄 1칸이 광석 1칸을 즉시(~4틱) 환원**하고 자신은 소모되며
연기(CO₂)를 낸다. 완료 시 **Molten Metal**(쇳물)이 되고 식으면 **Iron**으로 굳어 노 바닥에
고인다. **Limestone** 플럭스가 철 수율(0.70→0.95)을 올린다.

**핵심(용해·혼합 문제 해결):** ① 환원은 **발열 반응**이라 산물이 뜨거운 쇳물(≥1450°)로
나와 **가라앉는다** — 예전처럼 표면이 고체 철로 굳어 웅덩이를 봉해버리는 크러스트가 생기지
않는다. ② **광석에 직접 닿은 석탄은 연소하지 않고 환원제로만 작동**한다(환원 분위기) — 그래서
**석탄을 광석과 미리 섞거나 층으로 쌓아 가열해도 석탄이 먼저 타 없어지지 않고** 광석을 환원한다.
광석에서 한 칸 이상 떨어진 석탄은 정상적으로 타서 열을 낸다. ③ **용융 웅덩이에 뿌린 석탄은
웅덩이 속으로 가라앉아 섞이므로** 표면만이 아니라 전체가 환원된다(가벼운 석탄을 강제로 저어
넣는 셈).

**두 가지 방법 모두 작동:** (A) 광석을 녹인 웅덩이에 석탄가루를 뿌리면 가라앉아 섞이며
환원(얕은 웅덩이 수율 ~65%), (B) 석탄가루와 광석을 미리 섞거나 층으로 쌓고 밖에서 가열하면
석탄이 타지 않고 녹은 광석을 환원(장입 방식, 수율 ~60%). 석회석을 더하면 ~87%.

**밀도 층화:** 석탄가루(5) < 슬래그(6) < 용융 철광석(7) < 쇳물(8). 뿌린 석탄은 웅덩이에
섞여 들어가고, 환원된 쇳물은 바닥으로, 슬래그는 그 위에 뜬다 — 노의 수직 존이 저절로 생긴다.

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Iron Ore | 7 | 적갈색 광석 가루. **850°↑**(일반 Fire로도 도달)이면 **용융 철광석**으로 녹는다. 녹기 전엔 그냥 낙하·퇴적하는 가루. 녹이는 것만으로는 철이 안 나온다 — 석탄이 있어야 환원된다. |
| Molten Iron Ore | 7 | 녹은 철광석(빛나는 주황 액체). 닿은 **석탄가루/석탄**이 즉시(~4틱) 환원(석탄 1칸=광석 1칸, 연기 배출) → 발열로 뜨거운 **Molten Metal**(수율 0.70, **Limestone** 인접 시 0.95, 실패분 Slag)이 되어 가라앉는다. 석탄 없이 **750° 아래로 식으면 Slag로 굳음**(실패 산물). 점성 액체로 흐른다. |
| Coal Powder | 5 | **부어서 쌓이는 석탄** — 고체 Coal(id 25)은 형태를 유지하지만, 이 가루는 낙하·퇴적해 광석과 섞거나 층으로 쌓을 수 있다. **광석·쇳물에 직접 닿으면 연소하지 않고 환원제로 작동**(먼저 타 없어지지 않음), 용융 웅덩이에 뿌리면 **가라앉아 섞인다**. 광석에서 떨어지면 정상 연소(스펙은 Coal과 동일). |
| Limestone | 5 | 선택적 **플럭스**(백악색 가루). 자체 로직 없음 — 인접 환원 중 용융 광석의 철 수율을 0.70→0.95로 올리고 소모된다(하소 연출). 없어도 제련은 되지만 슬래그가 섞인다. |
| Slag | 6 | 제련 폐기물. **고온(≥600°)이면 점성 액체로 흘러내리고**(Lava보다 걸쭉), 식으면 어는점(600°) 처리로 **회청색 크러스트로 응고**(별도 물질 없이 glow로 어두워짐 — 재가열하면 다시 흐름). 쇳물(8)보다 가벼워 위에 뜬다. 슬래그로 노를 지으면 노가 뜨거워질 때 녹아내린다. |

## 🛢️ 석유 (Petroleum)

원유(Crude Oil)를 **끓여서 분별증류**하는 계열. 핵심 사용법: **Stone/Iron으로 용기를
만들고 그 아래에서 일반 불(Fire)/용암으로 가열**하면, 벽을 타고 전도된 열이 원유를
데워 끓는점 순으로 유분이 갈라져 나온다. 원유·유분은 **불꽃이 직접 닿으면 잘 탄다**(정상
가연성). 다만 **간접적인 열만으로는 발화하지 않는다** — 불꽃이 없으면 원유는 발화점에
닿기 전에 아스팔트로 크래킹되고 유분은 재비등해 날아가 버리기 때문이다. 그래서 밀폐 용기를
아무리 뜨겁게 데워도 타지 않고 증류된다("불 직접 대면 연소, 벽 너머로 데우면 분별증류").

원유 셀은 온도대에 따라 **제자리에서 해당 유분 증기로 끓어** 위로 올라간다(묻힌 원유도
증기가 액체 사이로 기포처럼 상승하므로 배출구가 필요 없다): `150–205° 석유가스,
205–258° 휘발유, 258–315° 등유, 315–380° 경유`, 380° 이상이면 **아스팔트**(고체 타르)로
크래킹된다. 아래에서 가열하면 뜨거운 바닥→차가운 위의 온도 구배가 생겨 **높이별로
자동 분별**된다(무거운 유분·잔사유는 낮고 뜨겁게, 가벼운 유분·가스는 높고 차갑게). 증기는
올라가 응결해 액체가 되고, 응결된 유분이 다시 뜨거운 바닥으로 떨어지면 **재비등(reflux)**해
증기로 되올라가므로 스스로 순환한다. 액체 유분은 밀도가 달라 저절로 층을 이룬다(가벼운
것이 위): **휘발유(2.2) < 등유(2.35) < 경유(2.45) < 원유(2.6) < 물(3)**.

> 히트 브러시(+)로도 데울 수 있지만 온도 상승이 빨라 조절이 어렵다 — **Stone/Iron 용기 +
> 불 가열**이 열전도로 천천히 데워져 증류를 관찰하기 좋다.

> **렌더링:** 석유 계열 액체(원유·휘발유·등유·경유)는 배경 틴트를 샘플링하지 않고
> **단색으로 렌더링**되어 한 덩어리의 균일한 유체로 보인다(나머지 액체는 미세한 색 변화).
>
> **유류화재:** 물 위에 석유를 붓고 점화하면 **표면에서 계속 타는 기름불**이 된다 — 물이
> 불을 끄지도, 아래 물이 빠르게 증발하지도 않는다(물과 닿은 석유층은 소화·소진되지 않고
> 물을 가려 보호).

|---|---|---|

## ⚡ 전기 (Electric)

> **전기 세기 시스템:** 스파크는 이동하며 **세기가 감쇠**한다 — 금속(Iron·Mercury)에서는
> 온전히 유지, 니크롬에서는 조금씩(저항 — 감쇠분이 줄열이 된다), 소금물에서는 천천히,
> 물에서는 빠르게 약해져 몇 칸 못 가 소멸. 도체(금속·수은·니크롬·물·소금물)만 전기를
> 통하고 **그 외 물질은 절연체**로 완전 차단한다. 전기전도는 더
> 이상 연료·가연성 기체를 **점화하지 않지만**(인화성 제거) **폭약(TNT 등)은 여전히 기폭**한다.
> 물·소금물에 전기가 흐르면 **전기분해로 소량의 수소·산소를 생성**한다(낮은 확률).

| 물질 | 거동 / 상호작용 |
|---|---|
| Iron | 최고 열전도 + **전기 도체**(세기 유지). 1400°↑ 용융. |
| Nichrome | **전열선(저항 발열체)**. 전기 도체지만 약간의 저항이 있어 세기가 서서히 감쇠(~30칸). **전기가 지나갈 때마다 저항열(줄열)로 가열**되어 회색 합금이 벌겋게 달아오름 — 맨 선은 공기 중 **복사 방열**과 평형을 이뤄 ~750° 백열(전원 끊으면 몇 초 만에 식음), 물·모래 등에 **묻으면 방열이 막혀 최대 1050°까지** 오르며 그 이상 자가 가열은 포화해 **스스로는 절대 안 녹는다**(전열선의 본분). 실물처럼 **양 끝 단자는 발열하지 않는 콜드 엔드**(3칸 이상 깔아야 가운데가 달아오름)라 배선으로 열이 덜 새어나간다. 열전도율도 낮아(실제 니크롬처럼 철의 1/6) 줄열이 코일에 머문다. 300°↑면 인접 가연물을 무화염 점화. **내산성**(산 속 배선 가능 — Iron 은 녹는다). 외부 열원으로 1450°↑ 시 용융. |
| Lithium Battery | 주기적으로 인접 도체에 Spark 주입(full 세기) → 자가 구동 회로. **고온(200°↑)이면 열폭주 — 잠깐 불붙어 타다가 폭발**(일반 화염 열로도 기폭). 평상시엔 **완만한 자체 방열**(방열판/보호회로)이 있어 배선을 타고 오는 은근한 열(예: 니크롬 줄열의 역류)은 버티지만, 직화에는 여전히 순식간에 폭주한다. |
| LFP Battery | 안전한 배터리 화학 — 리튬 배터리와 **동일한 주기·세기로 전기 공급**하지만 **연소·폭발(열폭주)이 전혀 없다**. 대신 방폭은 아니어서 폭발 피해에는 일반 고체처럼 파괴되고, Blue Flame·Lava·Thermite 급 극고온(1400°↑, 철과 동일)에서는 녹아내린다(→Molten Metal). 화염 속·니크롬 히터 옆에 둘 전원. |
| (Spark) | 도체를 타고 전파하는 전기 펄스(매질별 세기 감쇠). 폭약 기폭(연료는 점화 안 함). 불응 처리로 역류 방지. |

| (Heat Ray) | 임계 우라늄이 뿜는 **초고온 붉은 광선**. 8방향 직진, Wall·경계에서 **랜덤 각도로 반사**(규칙적 궤적 없이 카오스하게 확산), 서로는 간섭 없이 투과. 우라늄은 파괴하지 않는다 — **고체 우라늄은 명중 즉시 멜트다운**(→녹은 우라늄)하고, 녹은 우라늄은 +800° 대량 가열 + 그 셀의 **붕괴 촉발**(가열+붕괴 가속)로 임계 풀을 광선이 스스로 갉아먹는다. 스치는 우라늄 침전물을 즉시 액화시켜 연쇄 점화가 파일 사이로 빠르게 번진다. 그 외 모든 물질(Diamond·물 포함)은 파괴 후 확률적으로 관통 직진 또는 반사하며, 충돌 지점 주변을 강하게 지져(+400°) 녹이고·끓이고·태운다. **반사할 때마다 수명을 더 소모**하므로 지형을 관통하며 날아가는 광선은 화면을 가로지르지만 구석·발생원에 갇혀 반복 반사하는 광선은 곧 소멸(뭉침 방지). 우라늄이 충분하면 화면 전체를 초기화 수준으로 밀어버린다. |

## 대표 상호작용 사슬

- **금속 주조:** Iron → (열) → Molten Metal → (냉각) → Iron. Water 부으면 Steam 으로 냉각.
- **제련(1단계 용해):** Iron Ore 를 일반 Fire(또는 Lava·Blue Flame·산소 송풍 석탄불)로 850°↑까지 데우면 **Molten Iron Ore**(용융 웅덩이)로 녹는다. 녹이기만 하고 두면 식어서 Slag(실패).
- **제련(2단계 환원·주조):** 용융 웅덩이에 **Coal Powder 를 뿌리면**(가라앉아 섞임) 또는 광석과 미리 섞어/층으로 쌓아 가열하면(석탄이 타지 않고 환원) → 발열 반응으로 뜨거운 Molten Metal 이 되어 바닥에 고이고, 식으면 Iron. 슬래그(위)와 쇳물(아래)이 층으로 갈린다. Limestone 을 더하면 철 수율↑.
- **유리·소금 용융:** Sand → Molten Glass → Glass, Salt → Molten Salt → Salt (같은 상전이 패턴).
- **전기 기폭:** Lithium Battery → Iron/Mercury 배선 → Spark(세기 감쇠) → 아크 불꽃 → Gunpowder/TNT 기폭(연료는 점화 안 함).
- **전기분해:** 물·소금물에 전기가 흐르면 소량의 수소·산소 생성(수소는 산소와 만나 물로).
- **전기 곤로/주전자:** Battery → Iron 배선 → **Nichrome 코일**: 전기가 흐를 때마다 줄열로 달아올라(글로우) 물을 끓이고(수조 바닥 코일 = 전기 주전자) 소금을 녹이고 기름·나무를 무화염 점화 — 불 없는 전기 히터. 물이 다 끓어 마르면(boil-dry) 코일이 1000°+ 까지 치솟으니 리튬 배터리 배선을 짧게 붙이면 위험하다 — 극한 환경 전원은 열폭주 없는 **LFP Battery**로.
- **소화·중화:** Soda 를 화재에 뿌리면 불꽃 소화 + 타는 연료 냉각 진화(가루 소화기), Acid 웅덩이에 부으면 거품을 내며 Saltwater 로 중화.
- **도화선 폭파:** Fire → Fuse(지연) → TNT 연쇄.
- **냉각:** Dry Ice(승화해 **차가운 CO₂ 안개**를 뿜음) / LN₂(-196°) 가 물을 얼리고 불을 끔.
- **질식 소화:** CO₂(무거운 기체)가 바닥에 깔려 불을 덮어 끔 — Dry Ice 를 태우는 불 옆에 두면 CO₂ 안개로 자가 소화.
- **절단:** Thermite 점화 → 돌·철·유리를 뚫으며 녹임(붙어도 계속 낙하). 절단 후엔 뜨거운 용융 잔재가 식으며 잔류물 없이 서서히 소멸.
- **연소·제조수:** 불 → Hydrogen(가연성 기체) 연소, Oxygen 과 만나면 물(수증기) 생성 — 폭발이 아니라 조용한 연소.
- **유류화재:** 물 위에 석유(원유·휘발유 등)를 붓고 점화 → 표면에서 계속 타는 기름불, 아래 물은 증발 억제.
- **불멸 구조:** Diamond 는 모든 폭발·고온·저온·산에 면역이면서 열은 전도.
- **생태계·재배:** Water + Dirt → Mud, **Seed 를 젖은 흙에 심으면 발아 → Plant** 가 물을 빨아올려 위로 자람(물이 마르면 멈춤), Virus 가 유기물을 감염.
- **발효:** Yeast + (Sugar/Honey) + Water → **Alcohol + CO₂** — 만들어진 알코올은 그대로 연료 사슬로 이어짐.
- **나트륨 반응:** Sodium 을 물에 던지면 불꽃+수소 화구, **뭉쳐 쌓으면 물 접촉 시 덩어리째 폭발**(양에 비례).
- **자연발화:** White Phosphorus 는 공기에 노출되면 스스로 불붙어 백연을 뿜고, 물에 담그면 꺼진다.
- **경화:** Resin 이 시간이 지나 굳어 Amber(호박)가 됨. Aerogel 로 열을 완전히 차단한 실험 격벽.
- **소스/싱크:** Clone(무한 생성) ↔ Void(무한 삭제) 컨베이어.
- **원자로/멜트다운:** Uranium 덩어리 자가 발열 → 물 냉각(Steam 배출)으로 운전 ↔ 냉각 중단 시 1500° 멜트다운 → Molten Uranium(물처럼 흐르는 최고 밀도 액체) → 재냉각하면 고체 복귀, 방치하면 2000° 임계 → Heat Ray 방사로 화면 전체 소거.
