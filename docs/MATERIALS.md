# 물질 도감 (Materials)

전체 물질과 핵심 상호작용 요약. 팔레트는 아래 카테고리 탭으로 묶인다(각 물질의
`category` 필드, 생략 시 `phase` 에서 자동 유도).

> `Eraser`(지우개), `Ember`(불씨), `Spark`(스파크), `Heat Ray`(열선)는 팔레트에
> 없다: 폭발 파편/전기 펄스/임계 열선으로 시뮬레이션이 스스로 생성하는 일시적
> 입자다.

## 🪨 고체 (Solid)

| 물질 | 거동 / 상호작용 |
|---|---|
| Wall | 파괴 불가 경계. 폭발·산·Void 모두 못 뚫음. |
| Stone | 정적 장벽. 1100°↑ → Lava 로 용융. |
| Glass | 투명. 내산성. 1250°↑ → Molten Glass 로 재용융. |
| Concrete | Cement + 물이 굳은 결과물. 강하지만 폭발·산에 취약. |
| Diamond | 최고 열전도. **모든 폭발·고온·저온·산에 면역**. 방폭·전열 구조물. 유일한 예외: 우라늄 임계 열선(Heat Ray)은 다이아몬드도 뚫는다. |

## 🏖️ 가루 (Powder)

| 물질 | 거동 / 상호작용 |
|---|---|
| Sand | 낙하·퇴적. 1250°↑ → Molten Glass. |
| Salt | 물에 용해 → Saltwater. 초고온(800°↑) → 용융염(Molten Salt). |
| Dirt | 물 흡수 → Mud. Moss 가 덮는 지반. |
| Cement | 물 접촉 → Concrete 로 경화(물 소모). |
| Ash | 연소 잔재. 아주 가벼워 물에 뜬다. 가루가 흩날리듯 좌우로 흔들리며 천천히 떨어진다. |

## 💧 액체 (Liquid)

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Water | 3 | 100° 끓음→Steam, 0°↓ 얼음/눈. |
| Saltwater | 4 | 물보다 무겁게 가라앉음. 끓으면 소금 석출. |
| Acid | 3 | 비내산성 고체·가루 부식. 끓으면 Acid Vapor. |
| Mercury | 9 | 가장 무거운 일반 액체(Molten Uranium(10) 다음). **전기 도체**(스파크 전달). |
| Honey | 3.5 | 점성 높아 천천히 흐름. 느리게 타는 연료. |
| Alcohol | 1.9 | 가장 가벼운 연료. 순식간에 옮겨붙어 확 탄다. |
| Mud | 5.5 | 젖은 흙. 마르면 Dirt 로 굳음. |

## 💨 기체 (Gas)

| 물질 | 거동 / 상호작용 |
|---|---|
| Steam | 상승 후 응결 → Water. |
| Smoke | 상승 후 확률 소멸. |
| Acid Vapor | 상승하며 부식, 응결 → Acid. |
| Oxygen | 불 접촉 시 자신도 불로 확 번짐(가속제). 수소와 함께 연소하면 수증기(→물) 생성. |

## 🔥 불·열 (Fire / Heat)

| 물질 | 거동 / 상호작용 |
|---|---|
| Fire | 가연물 점화, 물에 소화. |
| Blue Flame | 초고온. 돌을 녹여 Lava 로. |
| Lava | 용융 암석. 식으면 Stone. |
| Molten Metal | Iron 이 녹은 것. 식으면 Iron. 밀도 8 액체. |
| Molten Glass | Sand 가 녹은 것. 식으면 Glass. |
| Molten Salt | Salt 가 녹은 것(800°↑). 식으면 Salt. |
| Coal / Wood / Sawdust | 고체 연료(느림→빠름 순: Coal < Wood < Sawdust). 액체 연료(원유·휘발유 등)는 아래 **석유** 탭 참고. |

## 🛢️ 석유 (Petroleum)

원유(Crude Oil)를 **끓여서 분별증류**하는 계열. 핵심 사용법: **Stone/Iron으로 용기를
만들고 그 아래에서 일반 불(Fire)/용암으로 가열**하면, 벽을 타고 전도된 열이 원유를
데워 끓는점 순으로 유분이 갈라져 나온다. 원유·유분은 **불꽃이 직접 닿으면 잘 탄다**(정상
가연성). 다만 **간접적인 열만으로는 발화하지 않는다** — 불꽃이 없으면 원유는 발화점에
닿기 전에 아스팔트로 크래킹되고 유분은 재비등해 날아가 버리기 때문이다. 그래서 밀폐 용기를
아무리 뜨겁게 데워도 타지 않고 증류된다("불 직접 대면 연소, 벽 너머로 데우면 분별증류").

원유 셀은 온도대에 따라 **제자리에서 해당 유분 증기로 끓어** 위로 올라간다(묻힌 원유도
증기가 액체 사이로 기포처럼 상승하므로 배출구가 필요 없다): `150–205° 석유가스,
205–258° 휘발유, 258–315° 등유, 315–380° 경유`, 380° 이상이면 **아스팔트**(고체 타르)로
크래킹된다. 아래에서 가열하면 뜨거운 바닥→차가운 위의 온도 구배가 생겨 **높이별로
자동 분별**된다(무거운 유분·잔사유는 낮고 뜨겁게, 가벼운 유분·가스는 높고 차갑게). 증기는
올라가 응결해 액체가 되고, 응결된 유분이 다시 뜨거운 바닥으로 떨어지면 **재비등(reflux)**해
증기로 되올라가므로 스스로 순환한다. 액체 유분은 밀도가 달라 저절로 층을 이룬다(가벼운
것이 위): **휘발유(2.2) < 등유(2.35) < 경유(2.45) < 원유(2.6) < 물(3)**.

> 히트 브러시(+)로도 데울 수 있지만 온도 상승이 빨라 조절이 어렵다 — **Stone/Iron 용기 +
> 불 가열**이 열전도로 천천히 데워져 증류를 관찰하기 좋다.

| 물질 | 밀도 | 거동 / 상호작용 |
|---|---|---|
| Crude Oil | 2.6 | 원료. 간접(전도) 가열 시 분별증류, 불꽃 직접 접촉 시 연소(두 번째로 빨리 탐). |
| Petroleum Gas | 0.8(기체) | 가장 먼저 끓어 나오는 경질 가스 산물. 상승·확산, 가연성(불 닿으면 확 붙음). |
| Petroleum Vapor | 1.2(기체) | 증류 중간 증기. 상승 후 응결해 해당 유분(휘발유/등유/경유)이 됨. |
| LPG | 1(기체) | 폭굉 없이 **폭연(연소)만** 하는 가연성 기체. 불꽃 접촉/400°↑ 자연발화 시 셀 단위로 화염 전파(충격파·크레이터 없음). |
| Gasoline | 2.2 | 가장 가벼운 액체 유분. 가장 빨리 타는 연료. 200°↑ 재비등(reflux). |
| Kerosene | 2.35 | 중간 유분(등유). 가연성. 260°↑ 재비등. |
| Diesel | 2.45 | 무거운 액체 유분(경유). 액체 연료 중 가장 느리게 탐. 320°↑ 재비등. |
| Asphalt | (고체) | 잔사유(타르). 원유가 380°↑에서 크래킹된 고체 잔재. 바닥에 크러스트로 쌓임. 비가연성. |

## 💥 폭발 (Explosive)

| 물질 | 거동 / 상호작용 |
|---|---|
| Gunpowder | 가루 폭약(반경 8). 물에 젖으면 불발. 연쇄 기폭. |
| Nitro | 액체 폭약(반경 12). 물 무관. |
| Methane | 공기-연료 폭발 기체(반경 7). |
| Hydrogen | 가장 가볍고 격렬(반경 8). 저온 점화. Oxygen 과 만나면 대폭발 + 물(수증기) 생성. |
| TNT | 고체 대형 폭약(반경 14). 스파크/불/열로 기폭. |
| Fuse | 가장 느린 도화선. 불을 천천히 폭약까지 운반. |
| Thermite | 점화 → 초고온 융해(지형 절단, ~기존 시간) → 식으며 잔류물 없이 서서히 소멸(뜨거울 땐 천천히, 식을수록 빠르게). 열 글로우로 용융/냉각 시각화. |
| Blast | 폭발 충격파(직접 배치 가능). |

## ❄️ 냉각 (Cryo)

| 물질 | 거동 / 상호작용 |
|---|---|
| Ice | 고체 얼음. 녹으면 Water. |
| Snow | 가벼운 가루 눈. 흩날리듯 좌우로 흔들리며 천천히 떨어진다. 녹으면 Water, 더 얼면 Ice. |
| Liquid N₂ | -196°. 냉각원. 물을 얼리고 불을 끔. 데워지면 증발(공기). |
| Dry Ice | -78° 고체. 서서히 승화해 사라짐(따뜻할수록 빠르게). |

## ⚡ 전기 (Electric)

| 물질 | 거동 / 상호작용 |
|---|---|
| Iron | 최고 열전도 + **전기 도체**. 1400°↑ 용융. |
| Battery | 주기적으로 인접 도체에 Spark 주입 → 자가 구동 회로. |
| (Spark) | 도체를 타고 전파하는 전기 펄스. 폭약·연료 점화. 불응 처리로 역류 방지. |

## 🌱 생명 (Life)

| 물질 | 거동 / 상호작용 |
|---|---|
| Moss | 습기 있으면 인접 표면(돌·흙·나무·콘크리트)을 덮으며 번짐. 가연성. |
| Virus | 유기물·흙·물을 감염시켜 전파. 불·산·열로 치료. |

## ✨ 특수 (Exotic)

| 물질 | 거동 / 상호작용 |
|---|---|
| Clone | 처음 닿은 물질을 복제해 무한 방출(무한 소스). |
| Void | 닿는 모든 물질 삭제(무한 싱크, Wall 제외). |
| Antimatter | 물질 접촉 시 1:1 상호소멸 + 화염 섬광. |
| Uranium | 방사성 고체. 인접 우라늄 수만큼 자가 발열(연쇄반응) — 소량은 안전, 두꺼운 덩어리는 폭주. 물 냉각으로 원자로 운전 가능. 1500°↑ → Molten Uranium 으로 **멜트다운**. |
| Molten Uranium | 녹은 우라늄(코륨). 점성 없이 물처럼 흐르고 **모든 액체 중 최고 밀도(10)**. 발열은 더 빨라져 계속 가열되며, 1400°↓ 냉각 시 다시 고체 Uranium 으로 응고. **2000°(임계) 도달 시 표면에서 서서히 연소하며 Heat Ray 방출** — 즉발 폭발이 아니라 몇 초에 걸쳐 타들어간다. |
| (Heat Ray) | 임계 우라늄이 뿜는 **초고온 붉은 광선**. 8방향 직진, Wall·경계에서 **랜덤 각도로 반사**(규칙적 궤적 없이 카오스하게 확산), 서로는 간섭 없이 투과. 우라늄은 파괴 대신 대량 가열(+800°, 한두 발이면 냉간 블록도 멜트다운 — 연쇄 점화). 그 외 모든 물질(Diamond·물 포함)은 파괴 후 확률적으로 관통 직진 또는 반사하며, 충돌 지점 주변을 강하게 지져(+400°) 녹이고·끓이고·태운다. **반사할 때마다 수명을 더 소모**하므로 지형을 관통하며 날아가는 광선은 화면을 가로지르지만 구석·발생원에 갇혀 반복 반사하는 광선은 곧 소멸(뭉침 방지). 우라늄이 충분하면 화면 전체를 초기화 수준으로 밀어버린다. |

## 대표 상호작용 사슬

- **금속 주조:** Iron → (열) → Molten Metal → (냉각) → Iron. Water 부으면 Steam 으로 냉각.
- **유리·소금 용융:** Sand → Molten Glass → Glass, Salt → Molten Salt → Salt (같은 상전이 패턴).
- **전기 기폭:** Battery → Iron/Mercury 배선 → Spark → 아크 불꽃 → Gunpowder/TNT 기폭.
- **도화선 폭파:** Fire → Fuse(지연) → TNT 연쇄.
- **냉각:** Dry Ice(서서히 승화) / LN₂(-196°) 가 물을 얼리고 불을 끔.
- **절단:** Thermite 점화 → 돌·철·유리를 뚫으며 녹임(붙어도 계속 낙하). 절단 후엔 뜨거운 용융 잔재가 식으며 잔류물 없이 서서히 소멸.
- **폭연·제조수:** Spark/불 → Hydrogen+Oxygen 혼합 → 대폭발 + 수증기(→물) 생성.
- **불멸 구조:** Diamond 는 모든 폭발·고온·저온·산에 면역이면서 열은 전도.
- **생태계:** Water + Dirt → Mud, Moss 가 습한 표면을 덮음, Virus 가 유기물을 감염.
- **소스/싱크:** Clone(무한 생성) ↔ Void(무한 삭제) 컨베이어.
- **원자로/멜트다운:** Uranium 덩어리 자가 발열 → 물 냉각(Steam 배출)으로 운전 ↔ 냉각 중단 시 1500° 멜트다운 → Molten Uranium(물처럼 흐르는 최고 밀도 액체) → 재냉각하면 고체 복귀, 방치하면 2000° 임계 → Heat Ray 방사로 화면 전체 소거.
